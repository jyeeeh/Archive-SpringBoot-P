## 프로젝트 소개

### 👨‍🏫 그룹 스터디를 개설하여 학습하는 공간
- 학생들이 스터디 모임을 형성하여 과제를 인증하고, 학업 성취도를 향상하는 데에 도움을 줍니다.
  ### 🔍 특징
  - 스터디 인증 기능을 통해 학생의 공부 과정을 기록합니다.
  - LMS 시스템을 통해 학습을 자기 주도적으로 관리할 수 있도록 도움을 줍니다.
